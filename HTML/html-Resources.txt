游릴 Core HTML Tags :
    游릴 DOCTYPE Declaration:
        - This declaration defines the document type and version of HTML
    游릴 HTML Element:
        - This is the root element of an HTML page
    游릴 Head Section:
        - Contains meta-information about the document, like character encoding, title, linked CSS files, and scripts.
    游릴 Body Section:
        - Contains the contents of the HTML document.
        - This is where you place the content that will be visible to users.
    游릴 Comments
        - Comments are not displayed in the browser, but they can help document your HTML source code

游릴 List Tags :
    游릴 <ul>

    游릴 <ol>

游릴 Link Tags :
    游릴 External links:
        - These are links that lead to other websites. The target="_blank"  attribute opens the link in a new tab or window.
    游릴 Internal links:
        -These links lead to different sections within the same web page.
    游릴 Email link:
        - The mailto: protocol in the href attribute creates a link that opens the user's default email client with the address specified in the link.
    游릴 Telephone link:
        - The tel: protocol creates a link that, when clicked on a device capable of making phone calls, will start a call to the specified number.

游릴 Link Tags :
    游릴 <img>:
        - This tag is used for embedding an image in the HTML page. The src attribute specifies the path to the image file, and the alt attribute provides alternative text for the image if it cannot be displayed. The width attribute specifies the width of the image.
    游릴 <video>:
        - The <video> tag is used to embed video content. It can contain one or more <source> elements to specify multiple video sources for different formats. The controls attribute adds video controls, like play, pause, and volume.
    游릴 <audio>:
        - Similar to <video>, the <audio> tag is used for embedding audio content and can contain multiple <source> elements. The controls attribute adds audio controls.
    游릴 <iframe>:
        - This tag is used to embed another HTML document within the current page.


游릴 Form Tags :
    游릴 <form> :
        - This tag is used to create the form. The action attribute specifies where the form data should be sent when it's submitted, and the method attribute specifies the HTTP method (GET or POST).
    游릴 <label>:
        - Defines a label for an <input> element.
    游릴 <input>:
        - This tag is a versatile form element. Depending on the type attribute, it can be a text field, password field, radio button, checkbox, etc.
    游릴 <textarea>:
        - Allows for multi-line text input. It's useful for longer, free-form text like comments or descriptions.
    游릴 <input type="submit">:
        - Defines a button for submitting the form.


游릴 Inter active Tags :
    游릴 <button>:
        - A clickable button.   
    游릴 <details> and <summary>:
        - These tags create a collapsible section. <details> is the container, and <summary> provides a clickable heading. Content placed within <details> but outside <summary> is hidden or shown when the user clicks on the <summary>.
    游릴 <dialog>:
        - Represents a dialog box or other interactive component. JavaScript is used to show and hide the dialog.
    游릴 <progress>:
        - Displays a progress bar. The value attribute specifies the current progress.
    游릴 <input type="range">:
        - Creates a slider for selecting a numerical value within a specified range.
    游릴 <select> and <option>:
        - Create a dropdown list. <select> defines the dropdown, and <option> elements represent the options within the dropdown.

游릴 Special Purpose Tags :
    游릴 <iframe>:

    游릴 <embed>:

    游릴 <object>:

    游릴 <svg>:

游릴 Scripting Tags :
